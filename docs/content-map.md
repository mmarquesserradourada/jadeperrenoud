# Content Map

- `content/site.json`
  - Brand, SEO, contact links.
- `content/navigation.json`
  - Menu and top CTA.
- `content/home.json`
  - Hero section text and image.
- `content/services.json`
  - Service cards.
- `content/blog.json`
  - Blog cards.
- `content/faq.json`
  - FAQ accordion.
- `content/footer.json`
  - Footer links and copyright.

After changes, run `npm run build` to regenerate `index.html`.
